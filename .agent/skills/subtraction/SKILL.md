# Subtraction Skill

This skill calculates the difference between two numbers.

## Usage

When the user asks to subtract one number from another or find their difference, execute the Python script `skill.py` (specifically the `execute` function or `SubtractionSkill` class) with the numbers as arguments `a` and `b`.

### Parameters

- `a` (number): The first number (minuend).
- `b` (number): The second number to subtract (subtrahend).

### Example

Input:
`a`: 50
`b`: 18

Output:
`32.0`
