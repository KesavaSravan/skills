# Addition Skill

## Purpose
The Addition skill takes two numeric inputs and returns their sum.

## Inputs
- **a**: First numeric input.
- **b**: Second numeric input.

## Outputs
- **result**: The sum of inputs a and b.

## Usage
This skill can be used to perform addition of two numbers.

## Example
python result = add(5, 3) print(result) # Outputs: 8
