# Addition Skill

This skill calculates the sum of two numbers.

## Usage

When the user asks to add two numbers or find their sum, execute the Python script `skill.py` (specifically the `execute` function or `AdditionSkill` class) with the numbers as arguments `a` and `b`.

### Parameters

- `a` (number): The first number to add.
- `b` (number): The second number to add.

### Example

Input:
`a`: 15
`b`: 27

Output:
`42.0`
